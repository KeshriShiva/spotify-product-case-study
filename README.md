# 🎵 Spotify Product Case Study

A product management case study analyzing Spotify's user experience, key challenges, business metrics, and proposing a new AI-powered feature to improve user engagement.

This case study was created to practice structured product thinking using a PM framework commonly used during Product Manager interviews.
